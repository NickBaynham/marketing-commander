# Prompts — Index

No prompt templates exist yet. Versioned prompt templates arrive in Phase 9
(prompt registry) and Phase 11 (campaign-brief and content-plan agent
contracts, referenced in traceability as AGC-BRIEF and AGC-PLAN).

Rules that will govern them (already decided):

- Prompt templates are versioned and immutable
  ([domain model](../../docs/product/domain-model.md), PromptVersion).
- Every generation records the exact prompt version used (BR-016).
- User-authored content enters prompts only in delimited untrusted-input
  slots ([Technical Design, Section 6](../../docs/architecture/technical-design.md)).
